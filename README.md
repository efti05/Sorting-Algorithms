A. Instructions to Run the Program--

1. Install a C++ compiler such as GCC or MinGW.
2. Save the source code file as `sorting.cpp`.
3. Open terminal or command prompt.
4. Compile the program using:
   g++ sorting.cpp -o sorting
5. Run the program:

   * On Linux/macOS: ./sorting
   * On Windows: sorting.exe

B. Required Dependencies--

* C++ Compiler (GCC / MinGW)
* Standard C++ Libraries: iostream, vector, ctime

C. System Requirements--

* Operating System: Windows, Linux, or macOS
* Minimum RAM: 4 GB (recommended for large datasets)
